gnotes-linux

Floating sticky notes for Linux — tray-first, Tauri 2, Nobara-tested.

Features

  • System tray: New Note + Quit
  • Transparent borderless note windows
  • Rich text, colors, pin, lock, collapse
  • SQLite persistence + session restore
  • AppImage and .deb packaging

Quick start (dev)

git clone https://github.com/davidthegnomad/gnotes-linux
cd gnotes-linux
npm install
cargo tauri dev

Requires WebKitGTK 4.1 — see LINUX.md.
